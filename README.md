# webxdc-svelte-shadcn-template

Svelte 5 + shadcn-svelte starter for building webxdc apps without re-learning the host API every time.

## Quick start

```sh
bun install
bun run dev
```

Build the web app:

```sh
bun run build
```

Build an `.xdc` package:

```sh
bun run build:xdc
```
